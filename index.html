Plugin para generar leads

Plugin diseñado para sincronizar leads desde un formulario con Google Sheets y amoCRM.

Instrucciones

Para hacer uso del plugin bastará con copiar y pegar el siguiente fragmento de código en la página en la que se desea utilizar el formulario.

          
<div id="root"></div>
<script>
  var SHEET_ID = '1WmmEvZdglTJnBwx5MG62e9rDKny_Q5u3ncfAIuR0lKk';
  var SEND_EMAIL = true;
  var SEND_PHONE = true;
</script>
<script defer="defer" src="https://digitalbusinessacademy05.github.io/static/js/main.e5e3f62d.js"></script>
<link href="https://digitalbusinessacademy05.github.io/static/css/main.355de385.css" rel="stylesheet">

Configuración

Como es posible observar, existen algunas opciones de configuración precedidas por la palabra "var", estas se utilizan de la siguiente manera:

  • SHEET_ID: Es el ID de la hoja de Google Sheets en la que se generarán los leads. Debe ir entre comillas simples o dobles.
  • SEND_EMAIL: Esta opción habilita el campo de correo en el formulario. Puede ser true o false, sin comillas.
  • SEND_EMAIL: Esta opción habilita el campo de teléfono en el formulario. Puede ser true o false, sin comillas.
